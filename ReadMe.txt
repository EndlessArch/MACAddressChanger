====================

MACAddressChanger

====================

다른거 필요없고
(맥 기준임)
1. 옵션 키 누른 채로 위쪽의 와이파이 아이콘 클릭,
   그리고 현재 연결된 네트워크에서 잠시 연결 해제해줍시다(차단된 그 공유기 아니면 패스).

2. 자신이 애용하느 터미널 프로그램 열고,
   ifconfig -a link 치고 현재 사용 중인 이더넷 디바이스르 찾아줍시다.
   (해킨 아니면 기본적으로 en0가 자신의 이더넷 장치임.)

3. sudo ifconfig en0 ether AA:BB:CC:DD:EE:FF
   ㄱㄱ
   
4. ifconfig -a link | grep ether 했을 때 이더넷 장치의 MAC 주소가 뜰 텐데,
   주소가 그대로라면 jmp 3(계속 해봐도 안바뀌면 뭐.. 알아서 잘 하십쇼)
   
5. 잘 바뀌었다면 다시 그 공유기에 연결 ㄱ
