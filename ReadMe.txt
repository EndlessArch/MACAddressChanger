====================

 MACAddressChanger

====================

인터넷 연결/해제하는 방법을 몰라서 그냥 텍스트로 방법만 씀.

따로 다운할거 없이,
(맥 기준)
1. 옵션 키 누른 채로 위쪽의 와이파이 아이콘 클릭,
   그리고 현재 연결된 네트워크에서 잠시 연결 해제(차단된 그 공유기 아니면 문제없음).

2. 자신이 애용하는 터미널 프로그램으로,
   ifconfig -a link 치고 현재 사용 중인 이더넷 디바이스를 찾아주자.
   (맥북은 기본적으로 en0가 자신의 이더넷 장치.)

3. sudo ifconfig en0 ether AA:BB:CC:DD:EE:FF
   그대로 복붙해서 쓰면 센스 0.
   
4. ifconfig -a link | grep ether 실행하면 이더넷 장치의 MAC 주소가 뜰 텐데,
   주소가 그대로라면 다시 시도.
   안되면 노답이고.
