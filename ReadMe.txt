====================

MACAddressChanger

====================

아직 맥에서만 실행해봤고 리눅스는 될지 잘 모르겠음(아마 배쉬 파일에서 이더넷 디바이스(en0)를 자신에게 맞는거 찾아서 바꿔주면 될거임).
당연히 윈도는 안됨. 윈도는 다른 프로그램 많으니

-------------------------

원하는 공유기에서 차단된 사람들은 이 프로그램 사용하면
아마 해결될거고(아닐 수도 있음)

-------------------------

이게 자주 안먹히는데
애플이 MAC 주소 spoofing 막으려고 한다네요.
어떤 사람들은 이걸 그냥 버그라고 하기도 하고.

sudo ifconfig en0 ether AA:BB:CC:DD:EE:FF

도배하다가

sudo ifconfig en0 down
sudo ifconfig en0 up

으로 랜카드 껐다 켰다 해주다 보면

ifconfig -a link

쳤을 때 en0의 MAC 주소가 바뀔 때가 있음.

실행:
sudo run.sh ㄱㄱ
